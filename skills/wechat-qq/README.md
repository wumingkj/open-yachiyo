# WeChat & QQ Sender

Windows 平台微信和 QQ 自动发消息工具。

## 快速开始

```bash
# 安装依赖
pip install pyautogui pygetwindow pyperclip pillow

# 微信发消息
python wechat_send_message.py 联系人 "消息内容"

# QQ 发消息
python qq_send_message.py 联系人 "消息内容"

# 截图保存到本地
python qq_capture_and_reply.py 群聊名称
```

## 功能

- ✅ 微信自动发消息
- ✅ QQ 自动发消息
- ✅ 支持颜文字 (๑•̀ㅂ•́)و✧
- ✅ 截图保存到本地

## 系统要求

- Windows 10/11
- Python 3.10+
- 微信/QQ 桌面版
