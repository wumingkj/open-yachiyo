# 微信/QQ 智能回复助手 v2.0 使用说明

## 🆕 更新内容

### v2.0 新增功能：
- ✅ **支持微信和 QQ** 双平台
- ✅ **自动检测** 当前打开的聊天窗口
- ✅ **智能截图** 聊天区域（根据平台自动调整）
- ✅ **区分平台特性**，优化识别效果

---

## 📱 微信 vs QQ 区别

| 功能 | 微信 | QQ |
|------|------|-----|
| **窗口检测** | 检测 "微信" 或 "WeChat" | 检测 "QQ" |
| **聊天区域** | 右侧（左侧联系人栏 280px） | 右侧（左侧联系人栏 250px） |
| **输入框位置** | 底部偏右 | 底部中间 |
| **OCR 难度** | 较高（可能有防截图） | 较低 |
| **消息显示** | 左右分布 | 左右分布 |

## 🚀 使用方法

### 基本使用：
```bash
# 自动检测平台并截图
python chat_assistant_v2.py

# 指定联系人（可选）
python chat_assistant_v2.py "someone"
```

### 使用流程：
1. **打开微信或 QQ**，进入聊天窗口
2. **运行脚本**，自动检测平台
3. **截图聊天区域**，保存到本地
4. **将截图发送给 AI 助手**（比如我）
5. **AI 分析聊天内容**，生成回复建议
6. **手动发送** 或 使用脚本自动发送

---

## ⚠️ 注意事项

### 微信特殊说明：
- 微信可能有**防截图机制**，部分版本无法截图
- 如果截图失败，尝试：
  - 使用微信 PC 版（非网页版）
  - 关闭微信的"防截图"设置（如果有）
  - 使用其他截图工具辅助

### OCR 识别：
- 当前版本返回截图文件，需要**外部 OCR 处理**
- 可集成 pytesseract、百度 OCR、阿里云 OCR 等
- 或直接将截图发送给 AI 助手（如 Kimi、Claude）
